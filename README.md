## CookieCloud服务器端python实现

参考原始项目, 基于fastapi开发

## 使用方法

Windows双击运行打包出的exe程序即可启动

默认CookieCloud服务器地址 `http://localhost:10375/cookiecloud/`
或者将localhost替换为运行机器的ip地址

可配置项
- `COOKIE_CLOUD_PORT`: 制定服务器端口，默认为10375

## 开发环境

- `windows 10`
- `python 3.11.8`
- `requirements.txt`
